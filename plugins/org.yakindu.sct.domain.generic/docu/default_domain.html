YAKINDU Statechart Tools - Default Domain

YAKINDU Statechart Tools 'Default Domain' is a platform independent domain with a basic type system. Choose this domain if you are new to YAKINDU Statechart Tools or you want to develop a generic statemachine.

Default Type System

The 'Default Domain' integrated type system consists of the following simple types:
  • integer
  • real
  • boolean
  • string
  • void
